Phototagger

A javascript-based implementation of an online photo-tagging service, similar to Facebook's tagging functionality. It is similar to backbone in its implementation, but the framework is generated solely from javascript and jquery, without a pre-generated javascript framework. A basic Rails setup provides the persistence layer.
